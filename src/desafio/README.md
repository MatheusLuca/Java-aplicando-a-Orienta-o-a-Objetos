# Desafio - pratica de classes em Java

Esta pasta contem exercicios de pratica com classes e metodos em Java.

## Arquivos

- `Main.java`: executa todos os exemplos da pasta, instanciando objetos e chamando metodos.
- `Pessoa.java`: classe simples com metodo `exibirMensagem()`.
- `Calculadora.java`: retorna o dobro de um numero com `retornarDobro(double numero)`.
- `Musica.java`: modela uma musica com titulo, artista, ano e avaliacoes, incluindo calculo de media.
- `Carro.java`: representa um carro e calcula a idade com base no ano informado.
- `Aluno.java`: armazena nome e idade e retorna os dados formatados.

## Como executar

Compile e execute a classe `Main` do pacote `desafio` pelo terminal na raiz do projeto:

```bash
javac -d out src/desafio/*.java
java -cp out desafio.Main
```
